# EnglishBubble - 英语学习桌面助手 (Windows)

这是一个使用 Python + Tkinter 开发的 Windows 桌面英语学习工具，包含:

1. 前台常驻泡泡图标（可拖拽、透明效果）
2. 单词复习卡片（间隔复习、难度分组）
3. 双语查询（英文↔中文，自动识别语言）
4. 本地词典（ECDICT 170k+ 单词库，首次运行自动下载）
5. 一键添加复习卡片
6. 英文发音（pyttsx3 文本转语音）
7. **屏幕取词**（双击泡泡左侧，支持 OCR 识别 / 剪贴板监听）

## 运行环境

- Windows
- Python 3.9+

## 安装依赖

```bash
pip install -r requirements.txt
```

## 启动

```bash
python english_bubble_app.py
```

## 使用说明

1. 启动后会出现一个前台泡泡图标（左侧 "S" 为搜索，右侧 "L" 为复习）
2. 界面交互：
   - 左键单点泡泡左半部分（S）：打开查询窗口
   - 左键单点泡泡右半部分（L）：打开复习卡片
   - 左键双击泡泡左半部分（S）：启动屏幕取词
   - 右键点击泡泡：退出程序
   - 拖拽泡泡：自由移动位置

3. 查询功能：
   - 输入英文单词或中文词汇
   - 支持自动语言检测（英→中 / 中→英）
   - 本地查询（ECDICT 数据库）+ 在线翻译（MyMemory API）
   - 支持查看词性、发音、例句、CET-4/6 等级标签
   - 查询结果可复制编辑

4. 屏幕取词（双击泡泡左侧 💡 新功能）：
   - **OCR 模式**（需安装 Tesseract）：全屏选择 → 自动识别文字 → 查询
   - **剪贴板模式**（无需额外安装）：选择屏幕文本 Ctrl+C 复制 → 自动检测变化 → 查询

5. 复习卡片：
   - 单词卡片显示：英文 / 中文释义 / 发音 / 例句
   - 操作选项：
     - 记住了 → 自动延长复习周期
     - 不熟 → 立即进入待复习队列
   - 支持发音播放

### 单词复习模式

- **复习模式**：
   - 待复习：只显示到期的卡片
   - 不熟：只显示标记过"不熟"的卡片
   - 全部：显示所有保存的单词

- **卡片操作**：
   - 记住了 → 自动延后下次复习时间（间隔复习算法）
   - 不熟 → 标记为不熟，立即进入待复习
   - 发音 → 播放英文发音（pyttsx3）

### 查询结果与添加

- 查询结果自动显示：中文释义、发音、词性、例句、CET 等级等信息
- 点击"加入复习卡片"可直接保存到复习库
- 保存前可编辑中文释义，自定义学习内容

## 数据文件 & 数据库

- `vocab_data.json`：保存用户的复习单词卡片、学习进度
- `ecdict.db`：本地英文词典数据库（SQLite）
  - 首次运行时自动从 GitHub 下载 ECDICT CSV（约 63MB）
  - 转换为本地数据库（约 30MB），后续查询无需网络
  - 包含 170K+ 英文单词、发音、释义、CET-4/6/IELTS/GRE 等级标签
  - 查询速度快，支持精确匹配

### 初始化

- 首次运行时，如果检测到没有 `ecdict.db`，会自动后台下载 & 构建索引
- 过程中会在搜索窗口显示进度条
- 下载期间可以正常使用 MyMemory API 进行在线翻译（不需等待）

## 屏幕取词 - 进阶配置（可选）

### 使用剪贴板模式（推荐，无需安装）
- 默认模式，无额外配置
- 双击泡泡左侧 → 显示"📋 请选择文本并 Ctrl+C"提示
- 选择屏幕上的文本并复制 → 自动检测剪贴板变化 → 自动查询

### 启用 OCR 模式（可选）
如要使用 OCR（光学字符识别）实现自动识别屏幕文字：

1. **安装 Tesseract OCR**：
   - 下载 Windows 安装程序：https://github.com/UB-Mannheim/tesseract/wiki
   - 安装到默认位置
   - 或手动添加 Tesseract 到 PATH 环境变量

2. **验证安装**：
   ```bash
   tesseract --version
   ```

3. **使用 OCR**：
   - 双击泡泡左侧 → 全屏出现十字光标
   - 拖拽选择要识别的文字区域 → 自动提取文字 → 查询

## 技术栈 & 依赖

| 组件 | 用途 | 备注 |
|------|------|------|
| **Tkinter** | GUI 框架 | Python 内置 |
| **requests** | HTTP 请求 | 下载 ECDICT、翻译 API |
| **SQLite3** | 本地词典数据库 | Python 内置 |
| **pyttsx3** | 语音合成（发音） | 离线文本转语音 |
| **pypinyin** | 中文转拼音 | 显示汉字拼音 |
| **Pillow** | 图像处理 | 生成泡泡图标 |
| **pytesseract** | OCR 识别（可选） | 需配合 Tesseract 二进制 |

## 故障排查

### 问题：首次启动很慢
- **原因**：后台下载 ECDICT 数据库（约 5-15 分钟）
- **解决**：等待进度条完成，或查看控制台日志

### 问题：OCR 不工作
- **原因**：未安装 Tesseract 二进制
- **解决**：使用剪贴板模式（默认），或按上述步骤安装 Tesseract

### 问题：发音无声音
- **原因**：系统音量静音或音频设备问题
- **解决**：检查系统音量，测试其他应用音频

### 问题：无法连接 MyMemory API
- **原因**：网络连接问题或 API 不可用
- **解决**：检查网络，本地 ECDICT 数据库仍可使用
